# java-calculator
This is a calculator that I built in Java using user input.

## Features:
- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Exponents (^)
- Square roots (s)

## Extra:
- Tells if numbers are even or odd
- Handles divide by zero errors

I made this to practice loops, conditionals, and user input in Java. It's also easy to modify and expand.
